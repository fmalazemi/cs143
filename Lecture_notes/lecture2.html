01 · Today's Goals

Learning Objectives

By the end of this lecture, you will be able to:

  • Write conditions using comparison and logical operators.
  • Use a simple if statement to run code only when needed.
  • Use if-else to choose between two paths.
  • Use if-else if-else to choose among many paths.
  • Place one if inside another (nested if).
  • Avoid the most common decision-making mistakes.
02 · You Already Know This

Decisions Are Not New

In CS101 you made decisions in Python every day. The idea is the same in C++ — only the spelling and rules change.

Python (what you know)

if mark >= 60:
    print("Pass")
else:
    print("Fail")

C++ (what we learn today)

if (mark >= 60) {
    cout << "Pass" << endl;
} else {
    cout << "Fail" << endl;
}

Main changes: the condition needs ( ), the body needs { }, and there is no colon.

04 · The if Statement

The Simple if

#include <iostream>
using namespace std;

int main() {
    int age;
    cout << "Enter your age: ";
    cin >> age;

    // the body runs ONLY if the condition is true
    if (age >= 18) {
        cout << "You are an adult." << endl;
    }

    cout << "Done." << endl;  // always runs
    return 0;
}
If user types 20
Enter your age: 20 You are an adult. Done.
If user types 15
Enter your age: 15 Done.

Shape: if (condition) { ... }

05 · Two Paths

The if-else

#include <iostream>
using namespace std;

int main() {
    int number;
    cout << "Enter a number: ";
    cin >> number;

    // % gives the remainder; 0 means even
    if (number % 2 == 0) {
        cout << "Even" << endl;
    } else {
        cout << "Odd" << endl;
    }
    return 0;
}
If user types 4
Enter a number: 4 Even
If user types 7
Enter a number: 7 Odd

Idea: exactly one of the two blocks runs — never both.

06 · Many Paths

The if-else if-else Chain

#include <iostream>
using namespace std;

int main() {
    int mark;
    cin >> mark;

    // checked top to bottom; stops at first true
    if (mark >= 90) {
        cout << "Grade A" << endl;
    } else if (mark >= 80) {
        cout << "Grade B" << endl;
    } else if (mark >= 70) {
        cout << "Grade C" << endl;
    } else {
        cout << "Grade F" << endl;
    }
    return 0;
}
If mark = 85
Grade B
  • C++ uses else if (two words), not Python's elif.
  • The chain stops at the first true condition.
  • The final else catches everything else.
07 · An if Inside an if

Nested if

#include <iostream>
using namespace std;

int main() {
    int age, hasLicense;          // 1 = yes, 0 = no
    cin >> age >> hasLicense;

    if (age >= 18) {              // outer check
        if (hasLicense == 1) {   // inner check
            cout << "You can drive." << endl;
        } else {
            cout << "Get a license first." << endl;
        }
    } else {
        cout << "Too young to drive." << endl;
    }
    return 0;
}
If age = 20, hasLicense = 1
You can drive.
If age = 20, hasLicense = 0
Get a license first.

Idea: the inner if is only reached when the outer condition is true.

08 · Be Careful

Common Mistakes

  • Using = instead of ==
    if (x = 5) assigns 5; you want if (x == 5)
  • Semicolon after if
    if (x > 0); makes the body empty
  • Using elif
    C++ needs else if, two words
  • Forgetting parentheses
    the condition must be inside ( )
  • Missing braces
    only the first line is inside the if without { }
  • Wrong logic word
    use && / ||, not and / or style by habit

Habit: always use braces { }, even for one line. It prevents bugs.

Worked Example 1

Positive, Negative, or Zero

#include <iostream>
using namespace std;

int main() {
    int n;
    cout << "Enter a number: ";
    cin >> n;

    if (n > 0) {
        cout << "Positive" << endl;
    } else if (n < 0) {
        cout << "Negative" << endl;
    } else {
        cout << "Zero" << endl;
    }
    return 0;
}
If user types -5
Enter a number: -5 Negative

What it does: It checks the sign of the number using an if-else if-else chain. Exactly one message prints.

Focus: choosing among three paths.

Worked Example 2

The Larger of Two Numbers

#include <iostream>
using namespace std;

int main() {
    int a, b;
    cout << "Enter two numbers: ";
    cin >> a >> b;

    if (a > b) {
        cout << a << " is larger" << endl;
    } else if (b > a) {
        cout << b << " is larger" << endl;
    } else {
        cout << "They are equal" << endl;
    }
    return 0;
}
If user types 8 3
Enter two numbers: 8 3 8 is larger

What it does: It compares two numbers and reports which is larger, or that they are equal.

Focus: comparison operators inside a chain.

Worked Example 3

Sign and Parity (Nested)

#include <iostream>
using namespace std;

int main() {
    int n;
    cin >> n;

    if (n > 0) {                 // first: is it positive?
        if (n % 2 == 0) {        // nested: is it even?
            cout << "Positive and even" << endl;
        } else {
            cout << "Positive and odd" << endl;
        }
    } else {
        cout << "Not positive" << endl;
    }
    return 0;
}
If user types 6
Positive and even
If user types 9
Positive and odd

What it does: The inner if only runs when the number is positive — a clear use of nesting.

12 · Wrap Up

Summary

  • A condition is true or false; build it with ==, !=, <, >, &&, ||.
  • if runs a block only when the condition is true.
  • if-else chooses between two paths.
  • if-else if-else chooses among many paths, top to bottom.
  • Nested if places one decision inside another.
  • Always use ( ) for the condition and { } for the body.
