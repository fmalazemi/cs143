03 · The Big Picture

What Is C++?

C++ is a general-purpose programming language. It is fast and is used for games, operating systems, browsers, and high-performance software.

  • It is a compiled language.
  • You must tell C++ the type of every variable.
  • Lines end with a semicolon ;.
  • Blocks of code are wrapped in braces { }.

Compiled vs. Interpreted

Python (interpreted): the computer reads and runs your code line by line, directly.

C++ (compiled): a tool called a compiler first translates all your code into a machine program. Then you run that program.

06 · Your First Program

"Hello, World!" in C++

#include <iostream>   // brings in input/output tools
using namespace std;  // lets us write cout, not std::cout

int main() {          // the program starts running here
    cout << "Hello, World!" << endl;  // print a line
    return 0;         // tell the system: finished OK
}
Expected Output
Hello, World!

What it does: It prints the text Hello, World! on the screen and then moves to a new line.

This is the traditional first program in almost every language.

08 · Output

Showing Output with cout

#include <iostream>
using namespace std;

int main() {
    // << sends each item to the screen, in order
    cout << "I am learning C++" << endl;
    cout << "The answer is " << 42 << endl;
    return 0;
}
Expected Output
I am learning C++ The answer is 42
  • << means "send this to cout".
  • You can join text and numbers in one line.
  • endl starts a new line.

Python compare: print("...") adds a new line for you. In C++ you add endl yourself.

09 · Input

Reading Input with cin

#include <iostream>
using namespace std;

int main() {
    int age;                    // a box for a whole number
    cout << "Enter your age: "; // ask the user
    cin >> age;                 // read typed value into age
    cout << "You are " << age << " years old." << endl;
    return 0;
}
Example Run (user types 20)
Enter your age: 20 You are 20 years old.
  • The arrows point the other way: cin >> age.
  • Think: data flows from cin into the variable.
  • You must declare the variable first (int age;).

Python compare: age = input() gives text; C++ cin >> age reads it directly as a number.

Worked Example 2

Double a Number

#include <iostream>
using namespace std;

int main() {
    int number;                       // store the user's number
    cout << "Enter a number: ";
    cin >> number;                     // read it from the keyboard
    cout << "Double is: " << number * 2 << endl;
    return 0;
}
Example Run (user types 5)
Enter a number: 5 Double is: 10

What it does: It reads one whole number, multiplies it by 2 with number * 2, and prints the result.

Focus: combining cin input with a small calculation inside cout.

Worked Example 3

Name and Age Together

#include <iostream>
#include <string>     // needed to use text (string)
using namespace std;

int main() {
    string name;       // a box for text
    int age;           // a box for a whole number
    cout << "Enter your name: ";
    cin >> name;
    cout << "Enter your age: ";
    cin >> age;
    cout << "Hello " << name << ", you are " << age << "!" << endl;
    return 0;
}
Example Run (types Sara, then 19)
Enter your name: Sara Enter your age: 19 Hello Sara, you are 19!

What it does: It reads text and a number, then builds one sentence.

Note: cin >> name reads one word only (it stops at a space).

10 · Be Careful

Common Beginner Mistakes

  • Missing semicolon — cout << "Hi"  needs a final ;
  • Wrong arrows — write cin >> x, not cin << x
  • Forgetting the include — #include <iostream> is required for cin/cout
  • No type given — write int x;, not just x;
  • Wrong capital letters — it is cout, not Cout or COUT
  • Using Python style — print(...) does not exist in C++

Tip: Read the first compiler error first. Later errors are often caused by the first one.

11 · Side by Side

Same Task in Both Languages

Python

name = input("Name: ")
print("Hello", name)

No types. No semicolons. print adds a new line.

C++

#include <iostream>
#include <string>
using namespace std;
int main() {
    string name;
    cout << "Name: ";
    cin >> name;
    cout << "Hello " << name << endl;
}

Takeaway: The idea is the same. C++ just asks you to be more exact.

14 · Wrap Up

Summary

  • C++ is a compiled language: write → compile → run.
  • Every program starts in int main() { ... }.
  • Lines end with ; and code blocks use braces { }.
  • cout << ... shows output; use endl for a new line.
  • cin >> ... reads input into a variable.
  • You must give each variable a type (int, double, string).
