Today's Goals

Learning Objectives

By the end of this lecture, you will be able to:

  • Explain what a stream is.
  • Write to a file with ofstream and read from one with ifstream.
  • Check that a file opened, and read it line by line.
  • Use istream / ostream so one function works on any stream.
  • Format output with setw, fixed, and setprecision.
  • Build strings with stringstream and to_string.
The Big Idea

Everything Is a Stream

A stream is a flow of data. You already use two: cout sends data out, and cin brings data in.

Console streams (known)

  • cout — output to the screen.
  • cin — input from the keyboard.

File streams (new)

  • ofstream — output to a file.
  • ifstream — input from a file.

The good news: files use the very same << and >> you already know.

Part A

File Input / Output

Reading from and writing to files with <fstream>.

Part A · The Picture

Streams Connect Your Program to Files

Part A · Writing

Writing to a File

#include <fstream>
using namespace std;

int main() {
    ofstream out("data.txt");   // open for writing

    out << "Hello" << endl;      // use << just like cout
    out << 42 << endl;

    out.close();                // close when done
    return 0;
}
File data.txt now holds
Hello 42
  • ofstream out("data.txt") opens (or creates) the file.
  • Write with <<, exactly like cout.
  • close() finishes and saves the file.
Worked Example 1

Saving a List to a File

#include <fstream>
using namespace std;

int main() {
    ofstream out("scores.txt");

    for (int i = 1; i <= 3; i++) {
        out << "Player " << i << ": " << i * 10 << endl;
    }

    out.close();
    return 0;
}
File scores.txt
Player 1: 10 Player 2: 20 Player 3: 30

What it does: a loop writes three lines into the file. Writing to a file is just like printing — only the destination changed.

Part A · Reading

Reading From a File

#include <fstream>
#include <iostream>
using namespace std;

int main() {
    ifstream in("data.txt");

    if (!in) {                   // did it open?
        cout << "Cannot open file";
        return 1;
    }

    string word;
    in >> word;                  // read one word
    cout << word << endl;
    in.close();
    return 0;
}
  • ifstream in("data.txt") opens the file for reading.
  • Always check it opened: if (!in).
  • Read with >>, exactly like cin.

A missing file is the most common error — checking first saves trouble.

Part A · Whole Lines

Reading Line by Line

#include <fstream>
#include <iostream>
using namespace std;

int main() {
    ifstream in("data.txt");
    string line;

    // getline reads a whole line each time
    while (getline(in, line)) {
        cout << line << endl;
    }

    in.close();
    return 0;
}
  • getline(in, line) reads a full line, spaces included.
  • The loop stops on its own at the end of the file.
  • Use >> for single words, getline for whole lines.

The same pattern works for numbers: while (in >> x).

Worked Example 2

Read a File and Number Its Lines

#include <fstream>
#include <iostream>
using namespace std;

int main() {
    ifstream in("notes.txt");
    string line;
    int n = 1;

    while (getline(in, line)) {
        cout << n << ": " << line << endl;
        n++;
    }
    in.close();
    return 0;
}
If notes.txt has two lines
1: hello world 2: goodbye

What it does: reads each line and prints it with a line number. The loop ends automatically at the end of the file.

Part A · One Function, Any Stream

istream and ostream

#include <fstream>
#include <iostream>
using namespace std;

// works with the screen OR a file
void writeReport(ostream& out) {
    out << "Report" << endl;
    out << "======" << endl;
}

int main() {
    writeReport(cout);          // to the screen
    ofstream f("report.txt");
    writeReport(f);             // to a file
    return 0;
}
  • cout and ofstream are both kinds of ostream.
  • cin and ifstream are both kinds of istream.
  • So a function taking ostream& works for either.

Write the code once; choose the destination later.

Part B

String Formatting

Shaping how output looks: widths, decimals, and building strings.

Part B · Tools

Formatting With <iomanip>

Manipulators from <iomanip> control how the next value is printed. setw(n) sets a field width and pads to fit.

Part B · Alignment

Widths and Alignment

#include <iostream>
#include <iomanip>
using namespace std;

int main() {
    cout << setw(8) << "Name"  << setw(6) << "Age" << endl;
    cout << setw(8) << "Sara"  << setw(6) << 20    << endl;
    cout << left << setw(8) << "Ali" << 22 << endl;
    return 0;
}
Expected Output
Name Age Sara 20 Ali 22
  • setw(n) sets the width of the next item only.
  • By default values are right-aligned.
  • left switches to left alignment.
Part B · Decimals

Controlling Decimal Places

#include <iostream>
#include <iomanip>
using namespace std;

int main() {
    double pi = 3.14159265;

    cout << fixed << setprecision(2);
    cout << pi << endl;        // 3.14

    cout << setprecision(4);
    cout << pi << endl;        // 3.1416
    return 0;
}
Expected Output
3.14 3.1416
  • fixed means "use plain decimal form".
  • setprecision(n) then sets the number of decimal places.
  • Great for money: fixed << setprecision(2).
Worked Example 3

A Neatly Formatted Table

#include <iostream>
#include <iomanip>
using namespace std;

int main() {
    cout << fixed << setprecision(2);
    cout << left  << setw(10) << "Item"
         << right << setw(8)  << "Price" << endl;
    cout << left  << setw(10) << "Apple"
         << right << setw(8)  << 1.5 << endl;
    cout << left  << setw(10) << "Bread"
         << right << setw(8)  << 12.0 << endl;
    return 0;
}
Expected Output
Item Price Apple 1.50 Bread 12.00

What it does: left/right with setw line up the columns, and fixed + setprecision(2) show prices to two decimals.

Part B · Building Strings

stringstream and to_string

#include <iostream>
#include <sstream>
using namespace std;

int main() {
    // build a string from pieces
    stringstream ss;
    ss << "Score: " << 95 << "/" << 100;
    string result = ss.str();
    cout << result << endl;     // Score: 95/100

    // quick number to string
    string s = to_string(42);
    cout << s << endl;          // 42
    return 0;
}
Expected Output
Score: 95/100 42
  • A stringstream works like cout, but builds a string.
  • ss.str() gives the finished text.
  • to_string(n) turns a number into a string.
Be Careful

Common Mistakes

File I/O

  • Not checking the file opened
    always test if (!in)
  • Wrong header
    file streams need <fstream>
  • Mixing >> and getline
    >> leaves the newline behind

Formatting

  • Expecting setw to stick
    it affects only the next item
  • setprecision without fixed
    then it counts significant digits, not decimals
  • Forgetting <iomanip>
    needed for setw and setprecision

Tip: a file stream also closes itself when it goes out of scope — its destructor does it.

Wrap Up

Summary

  • A stream is a flow of data; files use the same << and >> as the console.
  • ofstream writes to a file; ifstream reads from one; always check it opened.
  • getline reads whole lines; >> reads words or numbers.
  • ostream& / istream& let one function serve the console or a file.
  • setw, left/right, fixed, and setprecision shape how output looks.
  • stringstream and to_string build formatted strings.

Files and formatting let your programs save real data and present it cleanly.
