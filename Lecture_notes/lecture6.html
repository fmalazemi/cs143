02 · From Python Lists

You Already Used Lists

In CS101 a Python list held many values. A C++ array is similar, with two big differences: every element has the same type, and the size is fixed.

Python list

scores = [90, 85, 70]
print(scores[0])   # 90
scores.append(60)  # can grow

C++ array

int scores[3] = {90, 85, 70};
cout << scores[0];   // 90
// fixed size: cannot grow

Same idea: indexing starts at 0 in both languages.

Part A

Arrays

A fixed-size group of values of the same type, stored side by side.

03 · The Picture

What an Array Looks Like

An array is a row of numbered boxes. Each box holds one value; each box has an index that starts at 0.

04 · The Syntax

Declaring and Filling Arrays

// 1. declare with a size (empty boxes)
int scores[5];

// 2. declare AND give values
int scores[5] = {90, 85, 70, 60, 95};

// 3. let the size be counted for you
int marks[] = {7, 8, 9};   // size is 3

// put a value into one box
scores[0] = 100;
  • The type (int) applies to every box.
  • The size in [ ] must be a whole number known in advance.
  • Values inside { } fill the boxes in order.

Note: an array cannot change size later. Choose a size big enough for your needs.

05 · The Access Pattern

Reaching One Element

Write the array name, then the index in [ ]. The expression scores[2] means "the box at index 2".

06 · Under the Hood

How Arrays Live in Memory

An array's boxes sit side by side in memory, with no gaps. If each int takes 4 bytes, the addresses go up by 4 each time.

Why it matters: because boxes are evenly spaced, the computer can jump straight to scores[i] instantly — no searching.

07 · The Main Skill

Looping Through an Array

#include <iostream>
using namespace std;

int main() {
    int scores[5] = {90, 85, 70, 60, 95};

    // index goes 0, 1, 2, 3, 4
    for (int i = 0; i < 5; i++) {
        cout << "scores[" << i << "] = "
             << scores[i] << endl;
    }
    return 0;
}
Expected Output
scores[0] = 90 scores[1] = 85 scores[2] = 70 scores[3] = 60 scores[4] = 95

Pattern: start i at 0 and loop while i < size. The counter i becomes the index.

Worked Example 1 · Arrays

Sum and Average

#include <iostream>
using namespace std;

int main() {
    int a[5] = {10, 20, 30, 40, 50};
    int sum = 0;

    for (int i = 0; i < 5; i++) {
        sum += a[i];          // add each element
    }
    cout << "Sum = " << sum << endl;
    cout << "Average = " << sum / 5 << endl;
    return 0;
}
Expected Output
Sum = 150 Average = 30

What it does: It walks through every box, adding each value to a running total, then divides by the count.

Reuse: this is the accumulator pattern from the loops lecture.

Worked Example 2 · Arrays

Find the Largest Value

#include <iostream>
using namespace std;

int main() {
    int a[5] = {12, 45, 7, 99, 30};
    int biggest = a[0];        // start with the first

    for (int i = 1; i < 5; i++) {
        if (a[i] > biggest) {  // found a bigger one?
            biggest = a[i];
        }
    }
    cout << "Largest = " << biggest << endl;
    return 0;
}
Expected Output
Largest = 99

What it does: It assumes the first element is the largest, then checks every other element and updates when it finds a bigger one.

Note: the loop starts at 1 because index 0 is already the starting guess.

08 · Be Careful

Array Mistakes

  • Out of range
    for int a[5], valid indices are 0–4. a[5] is past the end
  • No safety net
    C++ does not warn you; it reads garbage or crashes
  • Starting at 1
    the first element is index 0, not 1
  • Wrong loop limit
    use i < size, not i <= size
  • Trying to grow it
    an array's size is fixed at creation
  • Comparing whole arrays
    a == b does not compare contents

Golden rule: the last valid index is always size − 1.

Part B

Strings

Text is just a sequence of characters — and C++ makes it easy with the string type.

09 · Text in C++

The string Type

A string holds text. You must include the string library first.

#include <iostream>
#include <string>      // needed for string
using namespace std;

int main() {
    string name = "Sara";
    cout << name << endl;       // Sara
    cout << name.length();     // 4
    return 0;
}

Python compare

# Python
name = "Sara"
print(len(name))   # 4

C++ uses name.length() where Python uses len(name). Both count the characters.

10 · Reading Text

Two Ways to Read a String

cin >> — one word

string word;
cin >> word;   // stops at a space
// input: Hello World
// word  = "Hello"

getline — whole line

string line;
getline(cin, line);  // reads spaces too
// input: Hello World
// line  = "Hello World"

Choose by need: use cin >> for a single word, and getline when the text may contain spaces (like a full name).

11 · A String Is Like an Array

Characters and Indexing

A string is a sequence of characters, each with an index from 0. You can read any character with [ ], just like an array.

12 · Joining and Walking

Combine and Loop Over Text

string first = "Ali";
string last  = "Hassan";

// join strings with +
string full = first + " " + last;
cout << full << endl;   // Ali Hassan

// walk through each character
int n = full.length();
for (int i = 0; i < n; i++) {
    cout << full[i] << "-";
}
Expected Output
Ali Hassan A-l-i- -H-a-s-s-a-n-

Joining: the + sign glues strings together (this is called concatenation).

Looping: store the length in n first, then use full[i] for each character.

Worked Example 3 · Strings

Count the Vowels

#include <iostream>
#include <string>
using namespace std;

int main() {
    string word;
    cin >> word;
    int count = 0;

    int n = word.length();
    for (int i = 0; i < n; i++) {
        char ch = word[i];
        if (ch=='a'||ch=='e'||ch=='i'||ch=='o'||ch=='u')
            count++;
    }
    cout << "Vowels: " << count << endl;
    return 0;
}
If user types "education"
Vowels: 5

What it does: It looks at each character and adds 1 to the counter whenever that character is a vowel.

Combines: a loop, string indexing, and an if condition.

Worked Example 4 · Strings

A Personalised Greeting

#include <iostream>
#include <string>
using namespace std;

int main() {
    string name;
    cout << "Enter your full name: ";
    getline(cin, name);        // read the whole line

    string message = "Welcome, " + name + "!";
    cout << message << endl;
    cout << "Your name has " << name.length()
         << " characters." << endl;
    return 0;
}
If user types "Sara Ali"
Welcome, Sara Ali! Your name has 8 characters.

What it does: It reads a full name (with the space), builds a message by joining strings, and reports the length.

Note: the space counts as a character, so "Sara Ali" has 8.

13 · Be Careful

String Mistakes

  • Forgetting the include
    you need #include <string>
  • cin only reads a word
    cin >> s stops at the first space
  • Index out of range
    valid indices are 0 to length() − 1
  • Single vs double quotes
    "A" is a string, 'A' is one character
  • Mixing cin and getline
    a leftover newline can make getline read an empty line
  • Comparing with numbers
    a string is text, even if it looks like "123"

17 · Wrap Up

Summary

  • An array stores many values of one type in numbered boxes, indexed from 0.
  • Access any element with name[index]; the last index is size − 1.
  • In memory, array boxes sit side by side; the address grows by the element size each step.
  • Loop with for (int i = 0; i < size; i++) to visit every element.
  • A string is a sequence of characters; use s.length() and s[i].
  • Use cin >> for one word and getline for a whole line.
