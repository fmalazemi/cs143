02 · You Already Know This

Nesting Is Not New

In CS101 you already wrote an if inside an if, and a loop inside a loop. C++ uses the same idea — with braces instead of indentation.

Python

for i in range(3):
    for j in range(3):
        print(i, j)

C++

for (int i = 0; i < 3; i++) {
    for (int j = 0; j < 3; j++) {
        cout << i << j << " ";
    }
}

The skill today: keeping the levels organised so the logic stays clear.

Part A

Nested if

A decision made after another decision has already been true.

03 · The Idea

What Is a Nested if?

A nested if is an if placed inside the body of another if. The inner test is only reached when the outer test is true.

Example question: "Are you old enough to drive? If yes, do you also have a license?"

04 · The Syntax

Writing a Nested if

#include <iostream>
using namespace std;

int main() {
    int age, license;          // license: 1 = yes, 0 = no
    cin >> age >> license;

    if (age >= 18) {           // outer test
        if (license == 1) {    // inner test (only if outer true)
            cout << "You can drive" << endl;
        } else {
            cout << "Get license" << endl;
        }
    } else {
        cout << "Too young" << endl;
    }
    return 0;
}
age=20, license=1
You can drive
age=20, license=0
Get license
age=15, license=1
Too young

Note: match each { with its }. Indentation helps you see the levels.

05 · Choose Wisely

Nested if vs. else-if Chain

Use nested if when…

  • The second test only makes sense after the first is true.
  • The tests check different things (age, then license).

Use else-if chain when…

  • You pick one option from a list.
  • The tests check the same value (a mark → A/B/C).

Warning: too much nesting becomes hard to read. If you go more than 2–3 levels deep, consider combining tests with && instead.

Worked Example 1 · Nested if

Pass, and Maybe Distinction

#include <iostream>
using namespace std;

int main() {
    int mark;
    cin >> mark;

    if (mark >= 50) {              // first: did they pass?
        cout << "Pass" << endl;
        if (mark >= 85) {         // only checked if passed
            cout << "With distinction!" << endl;
        }
    } else {
        cout << "Fail" << endl;
    }
    return 0;
}
If mark = 90
Pass With distinction!
If mark = 60
Pass
If mark = 40
Fail

Idea: the distinction test only runs for students who already passed.

Worked Example 2 · Nested if

Ticket Price by Age and Membership

#include <iostream>
using namespace std;

int main() {
    int age, member;          // member: 1 = yes, 0 = no
    cin >> age >> member;

    if (age < 13) {
        cout << "Child: free" << endl;
    } else {
        if (member == 1) {    // nested: only for non-children
            cout << "Member: 5 KD" << endl;
        } else {
            cout << "Adult: 10 KD" << endl;
        }
    }
    return 0;
}
age=10, member=0
Child: free
age=30, member=1
Member: 5 KD
age=30, member=0
Adult: 10 KD

Idea: membership only matters once we know the person is not a child.

06 · Be Careful

Nested if Mistakes

  • Mismatched braces
    each { needs a matching } — count them
  • Which else?
    an else pairs with the nearest unmatched if — use braces to be sure
  • Too deep
    4+ levels is hard to read

Cleaner with &&

// nested
if (age >= 18) {
    if (license == 1) {
        cout << "Drive";
    }
}
// same idea, flatter
if (age >= 18 && license == 1) {
    cout << "Drive";
}
Part B

Nested Loops

A loop inside a loop — the key tool for grids, tables, and patterns.

07 · The Idea

What Is a Nested Loop?

The inner loop runs all the way through for each single pass of the outer loop.

If the outer runs 3 times and the inner runs 4 times, the inner body runs 3 × 4 = 12 times in total.

Think: outer = rows, inner = columns.

08 · Syntax & Trace

Writing a Nested Loop

// outer uses i, inner uses j (different names!)
for (int i = 1; i <= 3; i++) {
    for (int j = 1; j <= 3; j++) {
        cout << i << j << " ";
    }
    cout << endl;   // new line after each row
}
Output
11 12 13 21 22 23 31 32 33
ijprints
11,2,311 12 13
21,2,321 22 23
31,2,331 32 33

For each value of i, the inner loop counts j from 1 to 3 completely. The endl after the inner loop ends each row.

Worked Example 3 · Nested loop

A Rectangle of Stars

#include <iostream>
using namespace std;

int main() {
    int rows = 3, cols = 5;
    for (int r = 1; r <= rows; r++) {   // each row
        for (int c = 1; c <= cols; c++) {   // each column
            cout << "*";
        }
        cout << endl;   // move to next line
    }
    return 0;
}
Expected Output
***** ***** *****

What it does: The outer loop chooses a row; the inner loop prints all 5 stars in that row.

Key: the endl belongs to the outer loop, not the inner one.

Worked Example 4 · Nested loop

A Multiplication Grid

#include <iostream>
using namespace std;

int main() {
    for (int i = 1; i <= 4; i++) {
        for (int j = 1; j <= 4; j++) {
            cout << i * j << "\t";  // \t = a tab gap
        }
        cout << endl;
    }
    return 0;
}
Expected Output
1 2 3 4 2 4 6 8 3 6 9 12 4 8 12 16

What it does: Each cell is i * j. The \t (tab) lines the numbers up into neat columns.

Worked Example 5 · Nested loop

A Right Triangle

#include <iostream>
using namespace std;

int main() {
    int n = 5;
    for (int i = 1; i <= n; i++) {
        // inner limit depends on the OUTER counter i
        for (int j = 1; j <= i; j++) {
            cout << "*";
        }
        cout << endl;
    }
    return 0;
}
Expected Output
* ** *** **** *****

The trick: the inner condition is j <= i. Row 1 prints 1 star, row 2 prints 2, and so on.

Idea: the inner loop can depend on the outer counter.

Worked Example 6 · Nested loop

A Number Triangle

#include <iostream>
using namespace std;

int main() {
    int n = 5;
    for (int i = 1; i <= n; i++) {
        for (int j = 1; j <= i; j++) {
            cout << j;   // print the column number
        }
        cout << endl;
    }
    return 0;
}
Expected Output
1 12 123 1234 12345

What it does: Same shape as the star triangle, but it prints j (the column number) instead of a star.

Compare: changing one line inside the inner loop changes the whole pattern.

09 · Be Careful

Nested Loop Mistakes

  • Same counter name
    the inner loop must use a different variable (use i and j)
  • endl in the wrong place
    inside the inner loop it breaks the line too often
  • Forgetting braces
    without { } only one line belongs to the loop
  • Wrong inner limit
    for triangles, use j <= i, not a fixed number
  • Very slow loops
    outer × inner can be huge; 1000 × 1000 = one million passes
  • Not resetting (while)
    with while, reset the inner counter each time
10 · Side by Side

Python vs. C++ Nesting

Python (indentation = levels)

for i in range(1, 4):
    for j in range(1, 4):
        print(i * j, end=" ")
    print()

C++ (braces = levels)

for (int i = 1; i < 4; i++) {
    for (int j = 1; j < 4; j++) {
        cout << i * j << " ";
    }
    cout << endl;
}

Takeaway: the structure is identical. In Python the indent shows the level; in C++ the braces do. Both have an outer and an inner loop.

13 · Wrap Up

Summary

  • A nested if places one decision inside another; the inner test runs only when the outer is true.
  • Use a nested if for follow-up questions; use an else-if chain to pick one option from a list.
  • A nested loop runs its inner loop fully for each outer pass (outer × inner total).
  • For shapes and grids: outer = rows, inner = columns.
  • The inner loop may depend on the outer counter (e.g. j <= i) to make triangles.
  • Use different counter names, mind your braces, and place endl with care.

Well done — you have now covered C++ basics, decisions, and loops.
