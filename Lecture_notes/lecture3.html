Learning Objectives

02 · You Already Know This

while Is Not New

You used while in CS101. The idea is the same: repeat a block as long as a condition is true.

Python (what you know)

i = 1
while i <= 3:
    print(i)
    i = i + 1

C++ (what we learn today)

int i = 1;
while (i <= 3) {
    cout << i << endl;
    i++;          // same as i = i + 1
}

Changes: the condition needs ( ), the body needs { }, and C++ offers the short form i++.

03 · The Idea

Why Do We Need Loops?

Without a loop, repeating work means copying the same line many times.

cout << 1 << endl;
cout << 2 << endl;
cout << 3 << endl;
// ... what about 1 to 1000?

A loop lets the computer repeat the work for us.

int i = 1;
while (i <= 1000) {
    cout << i << endl;
    i++;
}

A while loop is best when you do not know in advance how many times to repeat.

04 · The Three Parts

Anatomy of a while Loop

int i = 1;            // 1. START: set up a counter

while (i <= 5) {      // 2. CONDITION: keep going while true
    cout << i << endl; //    body: the repeated work
    i++;              // 3. UPDATE: move toward stopping
}
  • Start: create and set the counter before the loop.
  • Condition: a true/false test, checked before each pass.
  • Update: change the counter so the loop will end.

Warning: if you forget the update, the condition stays true forever — an infinite loop.

06 · A Common Pattern

Counting and Adding Up

#include <iostream>
using namespace std;

int main() {
    int n, i = 1, sum = 0;   // sum starts at 0
    cout << "Enter n: ";
    cin >> n;

    while (i <= n) {
        sum = sum + i;       // add i to the running total
        i++;
    }
    cout << "Sum = " << sum << endl;
    return 0;
}
If user types 5
Enter n: 5 Sum = 15

What it does: It adds 1 + 2 + 3 + 4 + 5. The variable sum is an accumulator — it grows each pass.

Key habit: set the accumulator to 0 before the loop.

07 · Unknown Count

Stop on a Special Value (Sentinel)

#include <iostream>
using namespace std;

int main() {
    int x, total = 0;
    cout << "Enter numbers (0 to stop):" << endl;

    cin >> x;               // read the first value
    while (x != 0) {        // keep going until 0
        total = total + x;
        cin >> x;           // read the next value
    }
    cout << "Total = " << total << endl;
    return 0;
}
User types 4, 6, 10, 0
Total = 20

What it does: It keeps adding numbers until the user enters 0. We do not know how many numbers there will be.

This is where while shines — the count is decided while the program runs.

08 · Two Helpers

break and continue

break — leave the loop now

int i = 1;
while (i <= 10) {
    if (i == 5) break;   // stop early
    cout << i << " ";
    i++;
}
1 2 3 4

continue — skip to next pass

int i = 0;
while (i < 5) {
    i++;
    if (i == 3) continue; // skip 3
    cout << i << " ";
}
1 2 4 5

Use sparingly. Clear conditions are usually easier to read than many breaks.

09 · Be Careful

Common Mistakes

  • Forgetting the update
    no i++ means the loop never ends (infinite loop)
  • Semicolon after while
    while (i < 5); makes an empty body that loops forever
  • Counter not started
    declare and set i before the loop
  • = instead of ==
    use == to compare in the condition
  • Off-by-one
    <= vs < changes how many times it runs
  • Updating the wrong way
    moving away from the stop value loops forever

If your program freezes, you probably made an infinite loop. Press Ctrl+C to stop it.

Worked Example 1

Count From 1 to 5

#include <iostream>
using namespace std;

int main() {
    int i = 1;            // start the counter
    while (i <= 5) {      // run while i is 5 or less
        cout << i << " ";
        i++;              // move toward the end
    }
    cout << endl;
    return 0;
}
Expected Output
1 2 3 4 5

What it does: It prints the numbers 1 through 5 on one line, separated by spaces.

Focus: the three parts working together — start, condition, update.

Worked Example 2

Countdown Then Liftoff

#include <iostream>
using namespace std;

int main() {
    int t = 5;
    while (t >= 1) {        // count DOWN this time
        cout << t << "... ";
        t--;               // subtract one each pass
    }
    cout << "Liftoff!" << endl;
    return 0;
}
Expected Output
5... 4... 3... 2... 1... Liftoff!

What it does: It counts down from 5 to 1, then prints a final message.

Focus: a loop can go down too — here the update is t--.

Worked Example 3

Average of Numbers (Sentinel)

#include <iostream>
using namespace std;

int main() {
    int x, count = 0, sum = 0;
    cin >> x;                  // first number
    while (x != -1) {          // -1 means stop
        sum = sum + x;
        count = count + 1;
        cin >> x;              // next number
    }
    if (count > 0)
        cout << "Average = " << sum / count << endl;
    else
        cout << "No numbers entered" << endl;
    return 0;
}
User types 10, 20, 30, -1
Average = 20

What it does: It adds numbers and counts them until -1, then prints the average.

Note: the if guards against dividing by zero when no numbers are entered.

13 · Wrap Up

Summary

  • A while loop repeats its body while a condition stays true.
  • Every loop needs three parts: start, condition, and update.
  • The condition is checked before each pass.
  • Use loops to count and to add up (accumulator starts at 0).
  • A sentinel value lets you stop when the count is unknown.
  • Forgetting the update creates an infinite loop — always move toward the stop.
